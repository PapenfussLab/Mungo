convertBlockSixFrameToGenomic(block, start, end)

source code 

Convenience function that takes block 6 frame coords (block,start,end), extracts the block start/end and frame and converts them to genomic coords

ie.

chrom.blockStart-blockEnd:frame aaStart aaEnd or chrom:blockStart-blockEnd:frame aaStart aaEnd --> chrom,blockStart,blockEnd,gStart,gEnd,strand

Parameters:
  • block - Block accession ("<name>.<blockStart>-<blockEnd>:<frame>")
  • start - Domain start
  • end - Domain end
Returns:
(chrom, blockStart, blockEnd, gStart, gEnd, strand)
